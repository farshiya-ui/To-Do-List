__________To-Do List App__________

Description :

A simple command-line To-Do List application built using Python. This application helps users manage their daily tasks by allowing them to add, view, and remove tasks. The tasks are stored in a text file, so they remain available even after the program is closed.

Features :

* View all tasks
* Add new tasks
* Remove existing tasks
* Store tasks permanently using file handling
* Simple and user-friendly menu interface

Technologies Used :

* Python
* File Handling
* Lists
* Functions
* Loops
* Conditional Statements

How to Run ?

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Open the terminal.
5. Run the program:
    -python todo.py


Menu Options :

1. View Tasks
2. Add Task
3. Remove Task
4. Exit
